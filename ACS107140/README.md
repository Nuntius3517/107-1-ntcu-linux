# 第一題
1. 輸入yum -y install wget來安裝wget的指令
![](https://i.imgur.com/FbS2giv.png)
2. 輸入 wget  https://raw.githubusercontent.com/ogre0403/107-1-ntcu-linux/master/resource/web.log ，就可下載apache web server的日誌檔。
![](https://i.imgur.com/vdFMT8M.png)
![](https://i.imgur.com/ddKtril.png)
3. 輸入指令"cat web.log | grep error"，就會顯示發生error的原因。
![](https://i.imgur.com/VaisHYT.png)
# 第二題
1. 輸入tar -jcv -f filename.tar.bz2 /var 2> tar-err.log，將/var的目錄打包壓縮，並將錯誤訊息儲存到tar-err.log。
