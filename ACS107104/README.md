1.去virtualBox官網下載virtualBox
(1)點進網站
(2)看到VirtualBox 5.2.18 platform packages點Windows hosts
2.去centos官網下載centos
(1)下載Minimal ISO
3.開啟virtualBox按新增
4.建立虛擬機器
(1)名稱:Centos7
(2)類型:Linux
(3)版本:Red Hat(64-bit)
(4)記憶體大小:1024MB
(5)硬碟:立即建立虛擬硬碟(C)
(6)硬碟檔案類型:(VirtualBox磁碟映像)
(7)實體硬碟中存放裝置:動態配置
(8)檔案位置和大小:Centos7;20GB
5.點virtualBox上面的設定值
6.點存放裝置，然後點選控制器下面的空
7.按旁邊光碟的圖案-選擇虛擬光碟檔案，從檔案管理裡找到你剛剛下載的Centos然後按確定
8.再按一次設定值，點選裡面的網路
9.點選介面卡2，按啟用網路卡，設定好後按確認
(1)附加到(A):僅限主機介面卡
10.按啟動
11.選擇語言(要按一下畫面→跳出一個畫面→按擷取)→點選系統-網路與主機名稱(N)→開啟兩個網路→點選系統-安裝目的地→按完成
12.按開始安裝→用戶建立(U)→裡面有兩個需要打勾的都勾→按進階→儲存變更→重新開機(R)
13.login打你用戶建立的帳號密碼(打密碼時不會顯示你打的東西)
