## 如何安裝Linux 
先打開VirtualBox後,再點新增,建立虛擬機器
![1](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/1.png?raw=true) 
建立後設定記憶體大小(大小皆可)
![2](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/2.png?raw=true) 
之後就點建立
![3](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/3.png?raw=true) 
點下一步
![4](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/4.png?raw=true) 
點下一步
![5](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/5.png?raw=true) 
按建立
![6](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/6.png?raw=true) 
建立完後點設定值,後點存放裝置,會出現一個cd圖示 空,點空,右方有個cd圖示,點下去後再點選擇虛擬光碟檔案,選擇CentOS下載檔,選擇好點確定
![7](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/7.png?raw=true)
![9](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/9.png?raw=true)  
之後點CentOS就會開始執行,會出現下圖的畫面
![8](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/8.png?raw=true) 
現在開始設定語言
![10](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/10.png?raw=true) 
語言選完會進入下圖的畫面,要點選安裝目的地,來磁碟分割
![12](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/12.png?raw=true) 
這時要選擇讓我自行配置磁碟分割的選項
![11](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/11.png?raw=true) 
之後會進入磁碟分割,按+加入新的掛載點
![13](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/13.png?raw=true)
要加入/ ./boot ,/home(大小自訂)
![14](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/14.png?raw=true)
分割完後點完成,再點接受變更
![15](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/15.png?raw=true) 
之後會出現用戶設定,設定Roots密碼和建立帳密
![16](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/16.png?raw=true) 
設定好後等下面藍色線跑完後,點重新開機
![17](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/17.png?raw=true) 
重新開機後會出現下圖的畫面,會要你輸入帳密(輸入密碼時是不會顯示輸入的內容)
![19](https://github.com/edmundabc/107-1-ntcu-linux/blob/HW-2/ACS107131/19.png?raw=true)  
輸入成功就表示你已經完成了
