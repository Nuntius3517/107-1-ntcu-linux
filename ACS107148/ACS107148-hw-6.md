### 第二小題
* 第一小段

const char *s=getenv("HOSTS_PATH");

    if(s==NULL){

	printf("getenv() return NULL);

	return 1;

}
首先：函式 getenv() 輸入環境變數的名稱字串，則輸出字元型別指標，如果名稱不存在於環境變數名單，則輸出null。

因此當我們找不到變數，指標s會被設定為null，並且映出錯誤訊息，然後回傳數值 1 。

第二小段
    printf("HOSTS_PATH :%s\n",(s!=NULL)?s:"getnev returned         NULL");

    printf("\n %s content is: \n",s);
分別代表正確輸出結果中的HOSTS_PATH : /etc/hosts和/etc/hosts content is:兩句話。 其中()?:的結果?真，輸出s。
